Guía de uso — Fogler Cap. 5–6
MRS-1 → MRS-2 (Batch) / MRS-3 (PFR) / MRS-4 (CSTR) → MRS-5 (Sweep) → MRS-6 (Métricas)
MRS-1 — Definir la red de reacciones
  • Species: lista de nombres separados por comas, e.g. CS, CP, CR
  • Rates: una expresión de velocidad neta rᵢ por línea, en el mismo orden que las especies. Usa sintaxis JS: *, **, Math.exp(). Los nombres de especies y constantes k son variables directas.
  • k values: k1=0.01, k2=0.005
  • C₀: concentraciones iniciales en mol/L, separadas por comas
  • Presiona Run Block → valida la red. Si aparece "Network OK", puedes continuar.
MRS-2 / MRS-3 — Batch & PFR
  • La red se enlaza automáticamente desde MRS-1 (dropdown "Use network from MRS-1").
  • Configura el rango de tiempo/volumen y el número de pasos RK4.
  • El gráfico SVG muestra Cᵢ vs t (o V). El marcador negro indica el óptimo de la especie objetivo.
  • Para PFR también ingresa v₀ (caudal volumétrico en L/s).
MRS-4 — CSTR (estado estacionario)
  • Ingresa τ = V/v₀ en segundos.
  • Resuelve el sistema Cᵢ₀ − Cᵢ + rᵢ(C)·τ = 0 por Newton-Raphson.
  • Salida: concentraciones de salida a ese τ específico.
MRS-5 — CSTR Parametric Sweep
  • Barre τ en el rango indicado y resuelve el CSTR en cada punto.
  • El gráfico muestra cómo cambia la composición de salida al variar el tamaño del reactor.
  • Identifica τ_opt que maximiza la especie objetivo.
MRS-6 — Selectividad & Rendimiento
  • Enlaza las concentraciones finales desde MRS-2, 3 o 4 (dropdown).
  • s_pairs: selectividad total S̃(D/U) → escribe CP/CW
  • y_pairs: rendimiento total Ỹ(D/A) → escribe CP/CS
  • Múltiples pares separados por coma: CP/CW, CR/CW
Ejemplo incluido por defecto
Red de 5 especies: CS → CP/CR (paralelas), CP → CR/CW (serie), CZ → CR.
Constantes del ejemplo de Fogler Cap. 6. Carga los defaults y presiona Run en MRS-1 para empezar.
